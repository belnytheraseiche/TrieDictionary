Remarks

This class stores keys in a DAWG, which is a data structure that compresses a standard Trie by merging common prefixes and suffixes to reduce redundancy.

Important Usage Notes:

  • Read-Only Structure: The dictionary is built once from a collection of keys and is effectively read-only afterward. The Add and Remove operations for keys are not supported and will throw a NotSupportedException.
